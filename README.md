## Smart Life Mall System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/738fc5da594e483e8daf7a59e0f3f8bc/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular Users, Administrators.

**Regular User Functions:**
1. Login
2. Registration
3. Homepage
4. Announcements
5. Products
6. Personal Center
7. Delivery Address
8. Product Favorites
9. Product Orders
10. Shopping Cart
11. Backend Management


**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Basic Data Management
4. Announcement Management
5. Product Management
6. Product Review Management
7. Product Message Management
8. Product Favorites Management
9. Product Order Management
10. Product Return Management
11. User Management
12. Banner Information
